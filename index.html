OMP Calculator Project
Backend Engineer on Project

My work on this project

My role in this project was to build a Spring Boot Backend Server to communicate with a ReactJS Frontend Server.... I worked alongside a frontend Engineer and at the beginning we decided to create a proof of concept Backend for Frontend (BFF) server to pass JSON data from server to server. We achieved this by using GraphQL to send queries using HTTP Requests.

Once we completed the proof of concept, we began building the real microservice. I created a Spring Boot server using Maven build tools and implemented both GraphQL and GraphiQL into the server. I then used SoapUI to test the XML Request and Response that we were provided with for the OMP-Calculator.

Once the response was verified I built the XML Document within the server using the Document Builder. Finally, I created HTTP Requests using the RestTemplate and passed XML Values to the ReactJS Frontend server where calculator functionality could take place.

Project Timeframe: 6 Weeks.

Remote Desktop Project
Server Creation and Configuration

My role

In this project I was tasked with configuring a Remote Desktop Connection within AIBs Intranet using Vagrant.... I worked alongside my mentor who was a Backend Engineer and we began by creating VMs using Vagrant. We created a Vagrantfile written in Ruby which specified details regarding SSH, Proxy and VM Creation. Once the VMs were built we booted them and SSH'd into them using Vagrant.

We then cloned all Microservices onto the VMs using SSH Keys and began building them on the CMD using Docker. We had to login to Nexus to achieve this and issue the command: docker-compose up to run the Microservices from within Vagrant. The microservices would then build and run in the CMD.

Once the development environemt was successfully set up we reported back to our Line Manager who then distributed connection details to all of the developers within AIB who could then connect remotely to this development environment and work on code while they were not at their workstations.

Project Timeframe: 2 Months.

Second Year Project
Java based Web Application using Play Framework

Summary

I worked on this project alongside 2 classmates when I was in Second Year in ITT Dublin.... We had 13 weeks to create a server side Java Based Web Application using an Oracle Database Backend and Play Framework to run the service. We began by making a list of user requirements and use cases so that we had a clear idea of the type of site we would be building. We ultimately decided to make a Video Game Store Site which would allow users to register, add games to a basket and checkout with their items.

Additional functionality included:
- Review a Game
- Use Loyalty Points
- Add, Update and Delete Games
- Add Credit Cards to Account

Firstly, we created a static HTML site before converting the site to Scala language within Play Framework. Each week we would meet with our Project Supervisor to discuss our progress and potential extras that we could add to the site. After 13 weeks we presented our finished product to a number of lecturers and got the chance to demo all of the functionality we added to the web app.

This was the first elapsed project I had worked on as part of a team and it gave me a great understanding of how to manage time and deadlines while also improving my communication skills.
